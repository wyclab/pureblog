## 肌酸（Creatine）

肌酸（Creatine）是健身界研究最透彻、公认最安全且有效的补剂之一。对于健身人士来说，它是一个非常优秀的“辅助工具”，但并非绝对的“必需品”。

简单来说，肌酸是你肌肉的“备用电池”。

人体在进行高强度、爆发性运动时，主要依靠ATP（三磷酸腺苷）来提供能量。但肌肉中储存的ATP非常少，几秒钟就会耗尽。肌酸在体内会转化为磷酸肌酸，它的核心作用就是**快速重新合成ATP**，让你的肌肉能持续输出能量。

人体自身（肝脏、肾脏）每天能合成大约1-2克肌酸，日常饮食中的红肉和鱼类也会提供一定量的肌酸，这足以应付日常生活的能量需求。

| **食物来源 (生鲜状态)**  | **近似肌酸含量 (克 / 每公斤)** |
| ------------------------ | ------------------------------ |
| **鲱鱼 (Herring)**       | 6.5 - 10.0 克                  |
| **猪肉 (Pork)**          | 5.0 克                         |
| **牛肉 (Beef)**          | 4.5 - 5.0 克                   |
| **鲑鱼/三文鱼 (Salmon)** | 4.5 克                         |
| **金枪鱼/吞拿鱼 (Tuna)** | 4.0 克                         |
| **鸡肉 (Chicken)**       | 3.4 - 4.0 克                   |
| **鳕鱼 (Cod)**           | 3.0 克                         |

虽然上述数字看起来不低，但在实际摄入时有一个非常重要的变量：**温度**。

- **热降解：** 当肉类和鱼类被加热烹调（煎、烤、煮、炸）时，食物中的一部分肌酸会降解并转化为一种叫做**肌酐（Creatinine）**的无用代谢物。
- **流失率：** 根据烹饪方式、温度和时间的不同，肌酸的流失率通常在 **20% 到 30%** 之间。如果炖煮时间过长，流失的比例还会更高（部分会溶解在汤汁中）。

为什么健身人士通常还是选择补剂？以牛肉为例，如果一个健身人士每天需要补充 **5克** 肌酸来维持肌肉中的饱和度，他不仅需要购买大约 1.2 到 1.5 公斤的生牛肉，还得把它们做熟并全部吃完。

这会带来几个现实问题：

1. **热量和脂肪超标：** 1.5公斤牛肉会带来巨大的热量和脂肪摄入。
2. **肠胃负担：** 每天消化这么多肉类对肠胃是非常大的挑战。
3. **经济成本：** 优质牛肉和鱼类的价格远高于一水肌酸补剂（一勺 5克 的一水肌酸成本极低）。

这就是为什么即便天然食物中含有肌酸，想要达到“提升运动表现”的剂量，直接使用提纯的肌酸粉依然是最经济、最高效的途径。

如果你决定开始使用肌酸，可以参考以下指南：

- **种类选择：** 首选**一水肌酸（Creatine Monohydrate）**。市面上有各种形式的肌酸（如盐酸肌酸、苹果酸肌酸等），但一水肌酸是目前科研证据最充分、效果最确切，且性价比最高的一种。不要为花哨的营销噱头支付溢价。

- **服用方法（两种主流策略）：**

  1. **平稳期法（推荐）：** 每天服用 **3-5克**，大约3-4周后，肌肉中的肌酸储备会达到饱和。这种方法温和，基本没有肠胃不适。
  2. **冲击期法（适合想快速见效的人）：** 前5-7天，每天服用 **20克**（分为4次，每次5克）以快速填满肌肉储备；之后每天维持 **3-5克**。

- **服用时机：** 肌酸的作用在于“储备”而非“即时刺激”（不像咖啡因），所以**什么时间吃都可以**，关键在于每天坚持。不过，一些研究表明，在训练后搭配碳水化合物或蛋白质一起服用，可能会因为胰岛素的升高而略微提高肌酸的吸收率。

- **注意事项：**

  - **多喝水：** 因为肌酸会把水分拉进肌肉细胞，如果不增加饮水量，可能会导致口干或轻微的肌肉抽筋。

  - **体重增加：** 初期体重可能会上涨1-2公斤，这完全是肌肉细胞储水造成的正常现象，并非长胖。

    

## 蛋白粉

蛋白粉（Protein Powder）本质上就是一种“高纯度、高效率、低热量”的便携食品。它不是什么长肌肉的“神药”，而是帮你轻松凑够日常蛋白质缺口的工具。对于有健身习惯的人来说，合理使用蛋白粉可以极大降低饮食准备的门槛。

补充蛋白粉的前提，是明确你每天的蛋白质总需求量。一般以体重（公斤）为基准来计算：

- **普通不锻炼人群：** 约 **0.8 - 1.0克** / 每公斤体重。日常饮食基本能满足。
- **耐力训练者（跑步、骑行）：** 约 **1.2 - 1.4克** / 每公斤体重。
- **力量与增肌训练者：** 约 **1.6 - 2.0克** / 每公斤体重。

**怎么补？** 假设你体重70公斤，目标是增肌，每天需要约140克蛋白质。如果你通过吃肉、蛋、奶只能吃够100克，那么剩下的40克缺口，就是蛋白粉发挥作用的地方（大约需要喝1.5到2勺）。如果你当天的饮食已经吃够了肉类，那就不需要喝。

蛋白粉的种类怎么选？市面上的蛋白粉琳琅满目，认准最主流的即可：

**分离乳清蛋白 (WPI - Whey Protein Isolate)：** 经过进一步过滤，蛋白质含量在90%以上，几乎去除了所有的脂肪和乳糖。吸收速度极快。不要选植物蛋白。



补充时机与实操细节

- **训练后补充（推荐）：** 在进行针对性的阻力训练（例如大重量的坐姿提踵或深蹲）后，肌肉纤维会发生微小撕裂。训练后30-60分钟内喝一勺（约20-30克蛋白质），能快速为肌肉修复提供游离氨基酸。

- **作为加餐或代餐：** 如果两餐之间间隔太长感到饥饿，或者早餐来不及吃高蛋白食物，可以用一勺蛋白粉搭配燕麦或全麦面包，既能提供饱腹感，又能平稳血糖。

- **单次摄入量：** 身体单次吸收蛋白质的效率是有限的。建议每次冲泡 **1到1.5勺（约25-40克蛋白质）** 即可，一次性喝太多不仅吸收不了，还会增加肾脏排泄的负担。

- **冲泡温度：** **切忌用开水冲泡。** 蛋白质遇高温会变性结块，不仅难喝，还会影响吸收。使用常温水、冷水或冰牛奶冲泡口感最佳。

  

## 维生素C

维生素C（抗坏血酸）可能是大众认知度最高、普及率最广的营养补充剂了。但与肌酸和蛋白粉不同，对于绝大多数现代人来说，维生素C的**日常饮食获取很容易**。

维生素C是一种水溶性维生素，人体无法自行合成，必须通过外界获取。成年人每天的推荐摄入量（RDA）大约只有 **100毫克**。

虽然日常饮食足够，但在以下特定场景或人群中，适当补充补剂是有益的：

1. **果蔬摄入极少的人：** 经常吃外卖、几乎不吃新鲜蔬菜和水果的人。
2. **高强度训练者/运动员：** 剧烈运动会产生大量的自由基（氧化应激）。适量的维生素C有助于抗氧化和缓解疲劳。此外，维C是合成胶原蛋白的关键，对关节、肌腱和韧带的修复至关重要。

几块钱一瓶的药字号（OTC）小白瓶，和几十上百块的保健品维C，在核心成分和作用上**没有任何区别**。**普通日常：** 随时可以吃，建议随餐服用，减少对胃粘膜的刺激。



## 钙、维生素D、K2

- **钙 (Calcium) —— 建筑原料（砖块）** 钙是你身体里含量最丰富的矿物质。它不仅是构成骨骼和牙齿的绝对主力，更是神经传导和**肌肉收缩**的核心触发器。你在做每一次大重量的坐姿提踵或深蹲时，肌肉纤维的收缩和舒张都需要钙离子的直接参与。
- **维生素D3 (Vitamin D3) —— 搬运工（运输车）** 当你吃下含钙的食物或补剂时，钙首先进入肠道。如果没有充足的维生素D3，你的肠道只能吸收大约 10% 到 15% 的钙。维生素D3的作用就是打开肠道吸收钙的通道，把钙从肠胃“搬运”到血液中。现代人由于长期在室内活动，加上防晒措施，绝大多数人都处于维生素D不足或缺乏的状态。 没有充足的维D，吃再多钙片也无法有效进入骨骼。
- **维生素K2 (Vitamin K2) —— 交通警察（导航员）** 这是最容易被忽视，却又最致命的一环。维生素D3只负责把钙拉进血液，但它不管钙去哪里。如果没有K2，血液中游离的钙很容易沉积在不该去的地方（比如血管壁、软组织或肾脏），导致动脉钙化或肾结石。**K2的作用是激活两种关键蛋白质：**
  1. **骨钙素 (Osteocalcin)：** 像吸尘器一样，把血液中的钙抓取并牢牢锁进骨骼和牙齿里。
  2. **基质Gla蛋白 (MGP)：** 像清道夫一样，把沉积在血管壁和软组织里的钙清理出去。

**“补钙 + 补D3 + 补K2”** 是目前公认的最安全、最高效的骨骼与心血管健康方案。

**1. 钙：尽量通过饮食获取**

- **来源：** 牛奶、酸奶、奶酪、老豆腐、深绿色蔬菜（如西兰花、羽衣甘蓝）。
- **补充建议：** 对成年人来说，每天需要约 800-1000 毫克钙。只要你每天喝奶且饮食正常，**通常不需要额外吃大剂量的钙片**。如果实在挑食需要补剂，首选**柠檬酸钙 (Calcium Citrate)**，它对胃酸依赖低，不易引起胀气和便秘。

**2. 维生素D3：强烈建议通过补剂获取**

- **来源：** 阳光照射皮肤合成、深海鱼、蛋黄。
- **补充建议：** 现代人室内活动多，防晒严密，靠晒太阳获取足够的D3极度不现实。建议直接购买纯度高的 D3 补剂。成年人日常维持剂量通常在每天 **1000 IU 到 2000 IU** 之间。

**3. 维生素K2：补充D3时的“必选项”**

- **来源：** 纳豆（含量极高）、发酵奶酪、动物肝脏。天然食物中K2的来源非常匮乏，且中国人的饮食习惯中很少包含这些食物。
- **补充建议：** 只要你开始单独补充维生素D3，就一定要搭配K2。
- **选购黄金法则：** 在购买时，请认准 **维生素K2 的 MK-7 形式 (Menaquinone-7)**。MK-7 在血液中的半衰期长达 72 小时，吸收率和生物利用度远高于其他形式（如MK-4）。



## 镁

镁是人体内参与300多种生化反应的必需矿物质，但它极易被压力、咖啡因消耗，且会随着汗液大量流失。

- **对健身与健康的作用：**
  - **解除肌肉紧绷：** 钙让肌肉收缩，而镁让肌肉放松。日常饮食中，像是一杯现打的花生豆浆就能为你提供不错的天然镁元素基础。但如果你经常进行高强度的肌肉破坏（例如高频次、大重量的提踵训练），随汗液流失的镁会大幅增加。缺乏镁最直观的后果，就是肌肉容易处于紧绷状态，甚至引发夜间小腿抽筋。
  - **改善睡眠质量：** 镁能调节中枢神经系统，帮助大脑和身体在睡前“断电”，进入深度睡眠。
- **补充建议：**
  - **避开氧化镁 (Magnesium Oxide)：** 这是市面上最廉价的镁，吸收率极低（约4%），吃多了容易拉肚子。
  - **推荐选择：甘氨酸镁 (Magnesium Glycinate)** 吸收率极高，对肠胃最温和，且助眠效果最好，建议在睡前30分钟服用；**柠檬酸镁 (Magnesium Citrate)** 吸收率也不错，适合运动后补充。

总而言之，如果你平时饮食已经比较均衡，优先考虑 **维生素D3** 和 **高纯度鱼油** 即可；如果训练强度大、睡眠质量不佳或肌肉容易抽筋，再加上 **镁** 会有立竿见影的改善。



## 鱼油

2026 年的医疗共识是，对于健康人群，非处方鱼油的疾病预防价值并不明确，还可能存在负面偏移。绝大多数临床指南并未推荐健康公众通过摄入非处方鱼油补充剂来预防原发性心血管疾病。

脂肪丰富的鱼类，例如三文鱼、鳗鱼、鲭鱼，富含多不饱和脂肪酸，想要吃“鱼油”的人大可买这些鱼来吃，每星期吃个两三次足矣。

欧米伽-3 脂肪酸可以被人体从 α-亚麻酸制造出来。核桃、亚麻等植物的种子含有 α-亚麻酸。婴幼儿奶粉之类添加的欧米伽-3 脂肪酸来自 Schizochytrium 属的 SAR 超类群海洋光合生物。市场上的非处方鱼油的纯度、成分（EPA 与 DHA 的比例）、氧化程度缺乏严格监管，这和临床试验使用的纯化处方级鱼油（例如大剂量的 Icosapent ethyl）不可比。
